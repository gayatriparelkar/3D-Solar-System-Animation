# 3D-Solar-System-Animation

A 3D visualization of the solar system created using Blender 4.5 as part of the Computer Graphics course (Semester V).
This project focuses on demonstrating planetary orbits, axial rotations, and detailed ring systems through smooth and continuous animation techniques.
